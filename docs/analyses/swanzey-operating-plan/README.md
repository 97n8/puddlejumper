# Swanzey Operating Plan — Budget · Forecast · Capital (PRIVATE)

One consolidated reference holding the Town's budget, a forecast, and the capital plan. Written to be
operational: on hire, the forecast becomes the working model, the reserve ladder becomes warrant
articles, and the 12-month plan becomes the calendar.

| File | Use |
|---|---|
| `Swanzey Operating Plan v0.1.pdf` | Read / print (PRIVATE) |
| `Swanzey Operating Plan v0.1.docx` | Edit / flip to the live operating doc on hire |

**Parts:** I — Budget (15-yr record, RSA 32 mechanics, the dead default lever, drivers, tax/revaluation) ·
II — Forecast (scenarios to 2031, approval risk, fund-balance guidance) · III — Capital (the 3/5 problem,
the 62%-other-money funding stack, the reserve ladder & tax glide, decision rules) · IV — Operating
Playbook (12-month plan, the annual municipal cycle, a dashboard).

Real figures from the Town's own annual reports (2011–2025), NH DRA, and local news; capital figures and
forecasts are clearly-labeled planning estimates, keyed to town-meeting year. Companions: Financial
Review, Capital Workbook (live model), Strategic Review, Interview Workbook.
