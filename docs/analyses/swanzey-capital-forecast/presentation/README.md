# Presentation Pack — Swanzey Capital Plan

Two ready-to-use deliverables for taking the capital forecast to the public and the ballot.

| File | What it is | How to use |
|---|---|---|
| [`deliberative-session-deck.md`](./deliberative-session-deck.md) | A 16-slide **Marp** deck for a pre-deliberative-session information night, with speaker notes | Render: `marp deliberative-session-deck.md --pdf` (or `--pptx`), or open with the Marp VS Code extension. Speaker notes are in HTML comments. |
| [`warrant-articles.md`](./warrant-articles.md) | **DRA-compliant model warrant articles** — reserve fund establishment, deposits, agents to expend, and the bridge-aid / water-SRF / CDBG authorizations | Edit numbers to the adopted CIP, run through **town counsel** + **DRA pre-review**, place on the warrant with recommendation tags |

## The flow they support

1. **Publish** the [Capital Forecast Book](../CAPITAL-FORECAST.md) and CIP.
2. **Present** the deck at an information night — *you cannot explain at the SB2 ballot box, so explain
   before.*
3. **Warrant** the model articles (Group A reserves first, then Group B leverage authorizations).
4. **Deliberative session** debates/amends; re-vote recommendations on anything amended (RSA 40:13).
5. **Ballot** in March — simple-majority reserve votes, no 3/5 bond to lose.

## Guardrails

- Every dollar figure is a **planning estimate** calibrated to Swanzey's published totals — confirm
  with engineering, NHDOT/NHDES, and DRA before presenting as final.
- Confirm the **West Swanzey water system's legal status** (town vs. precinct) before warranting B-2.
- Confirm the **bridge is on the NHDOT program list** and inspection is current before B-1.
- This is **model language and a model presentation** — not legal advice. Town counsel finalizes.
