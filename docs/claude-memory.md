# Claude Memory — PuddleJumper (GPR)

> Last updated: 2026-05-28. Load this at the start of every PJ session.

## What PuddleJumper is

Government Process Runtime. The fail-closed governance engine for PublicLogic.
Every municipal AI action passes through: **Evaluate → Register → Approve → Authorize → Dispatch**

**Live URL:** `publiclogic-puddlejumper.fly.dev` (Fly.io)
**Dev:** `pnpm dev:pj` → :3002 | `pnpm dev:vault` → :3003

                                                                                                                                                                                                                                                                                                                                                                                                  ## Seed Management
                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                  **Add a person:** edit `seed/people.yaml`, run `pnpm --filter @publiclogic/puddlejumper seed`
                                                                                                                                                                                                                                                                                                                                                                                                  **Change a role:** use `PATCH /api/org/identities/:id` — do NOT edit YAML (non-destructive by design)
                                                                                                                                                                                                                                                                                                                                                                                                  **Remove a person:** use `PATCH /api/org/identities/:id/deactivate` — do NOT just delete from YAML
                                                                                                                                                                                                                                                                                                                                                                                                  **Production bootstrap:** `PJ_DB_PATH=/app/data/pj.db pnpm --filter @publiclogic/puddlejumper seed`
