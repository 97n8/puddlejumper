## T — TRANSFER: does it survive turnover and replicate?

**Status:** This is the whole investment thesis and the deepest risk at once. **"All knowledge in McCall"** is a
critical succession risk for a 25-year, multi-site platform that must survive 2–3 leadership transitions. The
**parent-holdco → state-SPV** replication architecture (Nebraska LTC, Alabama LTC…) is exactly right — but it only
becomes real with a **documented module library / replication template** and an **institutional-continuity protocol**
(Rothschild/CFIR: stewards, evidence chains, knowledge transfer) so each next state inherits Michigan's governance
instead of restarting.

**Critical/High findings:** succession risk (T1); replication template (T2). **What closing it unlocks:** the actual
"to the stars" — a platform whose value compounds state by state because the governance, not any individual, carries
it forward. **Transfer is what turns one deal into a constellation.**
