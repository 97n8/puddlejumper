## Dimension-by-dimension: what an A requires

### 1. Thesis / positioning — A− → A  *(owner: PublicLogic)*
- **A requires:** one consistent narrative across every artifact; no conflicting numbers; a tight one-pager.
- **Do:** lock the byline and the "material upgrader / co-locate with recyclers" story; purge legacy EM&S framing;
  publish one approved one-pager. **Evidence:** single approved narrative doc; all decks reference it. *(Closeable now.)*

### 2. Feedstock control — B+ → A  *(owner: Robert / Garrett / National Salvage + ERR)*
- **A requires:** executed, dated supply agreements per active site with volume, term, and price (or tipping-fee terms).
- **Do:** convert the National Salvage and ERR relationships into **signed supply agreements** covering each site's
  required tonnage. **Evidence:** executed feedstock contracts in the data room.

### 3. Sites — B− → A  *(owner: Robert + PublicLogic on diligence)*
- **A requires:** locked site list; **signed site control** (lease/option/host co-location agreement) per active site;
  confirmed use-compatibility.
- **Do:** decide Coldwater vs Lincoln in the active set; execute host co-location/site-control docs (Coleman/Geocycle,
  Flint/ERR, Lincoln/National Energy); confirm Coldwater restrictive-covenant compatibility, Flint EJ posture, Coleman
  Part 169, Lincoln PPA interaction. **Evidence:** signed site control + a per-site permit/covenant compatibility memo.

### 4. Team / operators — B− → A  *(owner: Robert/Glen to retain; PublicLogic to coordinate)*
- **A requires:** a named, credentialed lead in **every** required discipline, with engagement letters.
- **Do:** retain (a) an **independent energy & emissions-modeling engineer**, (b) **securities counsel**, (c) a
  **financial-modeling/CFO-level** resource; publish an org chart + resumes. **Evidence:** signed engagement letters;
  bios in the data room.

### 5. Headline economics — D+ → A  *(owner: Glen [credits] + WasteWerx/Robert [opex]; PublicLogic builds the model)*
- **A requires:** one reconciled model where **opex is bottom-up and confirmed**, the **base case stands on
  contracted/market prices**, **credits are evidence-backed or quarantined as upside**, and the model passes an
  **independent third-party review**.
- **Do:** (1) WasteWerx confirms the full opex build-up beyond the 62-FTE labor; (2) Glen delivers RIN volume/price +
  §45Z rate + eligibility + registration status; (3) lock carbon-black price/offtake; (4) commission an independent
  model review. **Evidence:** reviewed model; base EBITDA defensible without credits; credit line items
  registration-backed. *(This is the heaviest lift and the lowest current grade.)*

### 6. Product offtake — C− → A  *(owner: Robert/Glen)*
- **A requires:** **executed** offtake agreements/MOUs for renewable diesel, SAF, and carbon black — named buyers,
  volumes, pricing, term — covering the modeled revenue.
- **Do:** convert verbal "uptake MOUs" into signed documents; resolve SAF/kerosene classification. **Evidence:**
  signed offtake set in the data room.

### 7. Deal / corporate structure — C → A  *(owner: securities counsel + Robert; PublicLogic coordinates)*
- **A requires:** parent holdco + state/site SPVs formed; executed operating agreements + board resolutions;
  LARA/foreign-entity registration; a securities-counsel-structured offering (e.g., Reg D) with the offering docs.
- **Do:** form/parent-paper the entities (close the VAULT-flagged authority/LARA gaps); engage securities counsel to
  structure and paper the raise. **Evidence:** entity chart, executed governance docs, counsel-issued offering structure.

### 8. Capital ask realism — C → A  *(owner: PublicLogic + Robert)*
- **A requires:** the per-site ask ties to the reconciled model; use-of-funds reconciles with no gap; company-raise
  allocation documented.
- **Do:** lock $/site to the model; fix the ~$1M use-of-funds gap; document the $2.5M company-raise split.
  **Evidence:** use-of-funds table that foots to the model. *(Closeable once #5 lands.)*

### 9. Diligence readiness / data room — C− → A  *(owner: PublicLogic)*
- **A requires:** a single source of truth + an organized data room; every cross-document number reconciled.
- **Do:** stand up the data room (model, contracts, entity docs, permits, resumes, disclosures); reconcile the
  3-version proforma, job counts, and community %. **Evidence:** indexed data room; one reconciled number set. *(Mostly
  closeable now.)*

### 10. Compliance posture — C+ → A  *(owner: securities counsel + PublicLogic)*
