LOGIC OS

The operations portal

Where your staff actually works. Microsoft 365 SSO out of the box. Case spaces, PRR management, document routing — all in one place. Runs in the browser. No install.

  • M365 / Entra ID single sign-on
  • SharePoint-backed document vault
  • Role-based case spaces
  • Public-facing status portal

PUDDLEJUMPER

The governance engine

Every action routes through here first. Intent is classified. Authority is checked. Approval chains run. Dispatch executes. Nothing happens outside the engine.

  • Fail-closed by default
  • Parallel + sequential approval chains
