# Metrics and Dashboard

## North Star
Number of municipalities with documented, compliant continuity packages in active use.

## Delivery Metrics
- Engagements active
- Pilot completion rate
- Time to deliver continuity package (weeks)
- Documentation completeness score (self-assessed checklist)

## Compliance Metrics
- Record types mapped to retention schedules
- Governance workflows documented
- Audit items resolved per quarter

## Growth Metrics
- Qualified leads per month
- Pilot conversion rate
- Retainer conversion rate
- Average engagement value

## Trust and Outcomes
- Client satisfaction (simple 1-5 score)
- Leadership confidence in continuity (1-5 score)
- Staff onboarding time reduced (weeks)
